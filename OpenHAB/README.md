# OpenHAB

After setting up an openHAB server in your preferred way you need to:

1. Copy over the files from this folder to your server
2. Copy over the UI from the files to your openHAB instance
    1. Navigate to OpenHAB's UI page management
    1. Create a new page
    1. Switch to the page's Code editor
    1. Copy and paste the YAML content from the files in this directory
